# 快速修复：会话建立失败

## 问题
恢复钱包时出现错误：**"会话建立失败，请稍后重试"**

## 原因
后端服务器（8787端口）未运行，且开发模式未启用。

## 解决方案（已完成）

### ✅ 步骤 1: 环境配置已创建
`.env` 文件已自动创建，包含以下配置：
```bash
VITE_ALLOW_DEV_SESSION=1  # 启用开发模式
```

### ⚠️ 步骤 2: **需要重启前端服务器**

**方式一：使用自动脚本（推荐）**
```bash
cd /home/xiaodong/文档/stardust/stardust-dapp
./重启开发服务器.sh
```

**方式二：手动重启**
```bash
# 1. 停止当前服务器 (在终端按 Ctrl+C)
# 2. 重新启动
cd /home/xiaodong/文档/stardust/stardust-dapp
npm run dev
```

### ✅ 步骤 3: 测试登录

1. 访问 http://localhost:5173
2. 点击"恢复钱包"
3. 填写助记词和密码
4. 点击"恢复钱包"按钮

### 📊 验证成功标志

打开浏览器控制台 (F12)，应该看到：

```
[session] createSession start {address: "5Grw..."}
[session] no sessionId, using dev fallback
[session] session created {expiresAt: ...}
🔍 表单状态: {canSubmit: true, ...}
```
