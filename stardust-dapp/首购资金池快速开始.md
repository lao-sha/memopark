# 首购资金池管理 - 快速开始指南

## 🚀 5分钟快速上手

### 做市商视角

#### 1️⃣ 申请成为做市商
```
访问: http://localhost:5173/#/otc/mm-apply
操作: 质押保证金 -> 填写配置 -> 提交申请 -> 等待审核
```

#### 2️⃣ 充值资金池
```
访问: http://localhost:5173/#/otc/market-maker-config
操作: 点击"充值首购资金池" -> 输入金额 -> 签名确认
建议: 首次充值 1000-5000 MEMO
```

#### 3️⃣ 管理资金池
```
访问: http://localhost:5173/#/first-purchase/pool
功能:
  - 查看余额、已用、冻结金额
  - 申请提取资金（7天冷却期）
  - 执行提取（冷却期结束后）
  - 取消提取申请
```

---

### 用户视角

#### 购买 MEMO（首购）
```
访问: http://localhost:5173/#/first-purchase
操作:
  1. 选择购买金额（50-100 MEMO）
  2. 可选填写推荐码（享9折）
  3. 确认做市商服务正常
  4. 创建订单并支付
  5. 支付成功后自动到账
```

## 💡 关键操作示例

### 提取资金完整流程

```
第1步: 申请提取
  访问: #/first-purchase/pool
  点击: "申请提取资金"
  输入: 提取金额（例如: 500 MEMO）
  选择: 是否暂停服务
  提交: 签名并发送交易
  结果: 资金被冻结，进入7天冷却期

第2步: 等待冷却期（7天）
  页面显示: 实时倒计时
  可选操作: 随时可以取消申请
  
第3步: 执行提取
  条件: 冷却期结束
  点击: "执行提取"
  结果: 资金转入您的账户
```

## ⚠️ 重要提示

### 资金安全
- ✅ 每个做市商有独立的派生账户
- ✅ 7天冷却期防止恶意提取
- ✅ 最小保留余额确保持续服务
- ✅ 治理委员会可介入异常情况

### 服务保障
- ⚠️ 可用余额低于100 MEMO时无法服务
- ⚠️ 提取申请会冻结资金
- ⚠️ 建议保持1000+ MEMO可用余额
- ⚠️ 余额不足时及时充值

### 常见参数
- 每次首购金额: **100 MEMO**
- 最小保留余额: **1000 MEMO**
- 提取冷却期: **7天**
- 做市商最小资金池: **1000 MEMO**
