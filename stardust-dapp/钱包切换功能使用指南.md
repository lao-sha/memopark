# 钱包切换功能使用指南

## 🎯 功能概述

钱包切换功能允许您在多个钱包账户之间快速切换，方便管理不同用途的资产。

## 📱 如何使用

### 1️⃣ 打开钱包切换

**方法一：从钱包管理页面**
1. 进入"我的钱包"
2. 点击"钱包管理"
3. 点击右上角的"🔄 切换"按钮

**方法二：快捷方式**
- 在钱包管理页面顶部，直接点击"切换"按钮

### 2️⃣ 查看钱包列表

切换弹窗会显示所有本地钱包：

```
┌─────────────────────────────────────┐
│ 🔄 切换钱包                         │
├─────────────────────────────────────┤
│ 钱包 5FHneW          ✅当前         │
│ 5FHneW...xyz                        │
│ 余额: 100.0000 MEMO                 │
├─────────────────────────────────────┤
│ 我的第二个钱包                       │
│ 5DTestKLiA...abc                    │
│ 余额: 50.0000 MEMO                  │
├─────────────────────────────────────┤
│ 测试钱包                             │
│ 5CiPPseX...def                      │
│ 余额: 0.0000 MEMO                   │
└─────────────────────────────────────┘
```

**钱包卡片信息：**
- 🏷️ **钱包别名**：您设置的钱包名称
- 📍 **钱包地址**：完整的钱包地址
- 💰 **余额**：实时查询的链上余额
- ✅ **当前标识**：蓝色边框 + "当前"徽章

### 3️⃣ 切换钱包

1. 在钱包列表中，点击您想切换的钱包卡片
2. 系统会自动：
   - ✅ 切换到选中的钱包
   - ✅ 关闭切换弹窗
   - ✅ 更新页面显示
   - ✅ 显示新钱包的余额和信息

**提示**：如果点击的是当前钱包，弹窗会直接关闭。

### 4️⃣ 创建新钱包

如果您需要创建新的钱包：

1. 在切换弹窗底部，点击"➕ 创建新钱包"
2. 按照引导完成：
   - 设置密码
   - 生成助记词
   - 备份助记词
   - 验证助记词
3. 创建成功后，自动切换到新钱包

### 5️⃣ 导入钱包

如果您有现有的助记词：

1. 在切换弹窗底部，点击"导入钱包"
2. 输入助记词和设置密码
3. 导入成功后，自动切换到导入的钱包

## 💡 使用场景

### 场景 1：工作和生活分离
- **工作钱包**：用于接收工资、处理业务
- **生活钱包**：用于日常消费、个人支出

### 场景 2：资产隔离
- **主钱包**：存放大额资产
- **交易钱包**：用于日常小额交易
- **测试钱包**：用于测试新功能

### 场景 3：多人共用设备
- **个人钱包 A**：第一个用户的钱包
- **个人钱包 B**：第二个用户的钱包
- 快速切换，互不干扰

## ⚠️ 注意事项

### 1. 钱包安全
- ✅ 所有钱包都加密存储在本地
- ✅ 切换钱包不需要重新输入密码
- ✅ 助记词始终加密，不会泄露
- ⚠️ 请妥善保管您的助记词和密码

### 2. 余额显示
- 💡 余额是实时从链上查询的
- 💡 首次打开可能需要几秒加载
- 💡 如果网络不稳定，可能显示"0.0000"
- 💡 可以点击刷新按钮重新查询

### 3. 钱包别名
- 💡 默认别名为"钱包 + 地址前6位"
- 💡 您可以在钱包设置中修改别名
- 💡 别名仅本地生效，不会上传

### 4. 切换限制
- ⚠️ 切换钱包会立即生效
- ⚠️ 正在进行的交易不会被影响
- ⚠️ 切换后需要重新授权某些操作

## 🔧 常见问题

### Q1: 为什么看不到切换按钮？
**A**: 请确保您在"钱包管理"页面，切换按钮在顶部右侧。

### Q2: 钱包列表是空的怎么办？
**A**: 点击"创建新钱包"或"导入钱包"来添加钱包。

### Q3: 余额显示"..."是什么意思？
**A**: 表示正在查询余额，请稍等几秒。

### Q4: 切换钱包后余额没更新？
**A**: 页面会自动更新，如果没有，请点击刷新按钮。

### Q5: 可以删除钱包吗？
**A**: 目前切换弹窗不支持删除，请在钱包设置中操作。

### Q6: 切换钱包需要密码吗？
**A**: 不需要，切换是即时的。只有在发送交易时才需要密码。

### Q7: 钱包别名可以修改吗？
**A**: 可以，在钱包设置中修改别名。

### Q8: 切换钱包会影响其他页面吗？
**A**: 会的，切换后所有页面都会使用新钱包。
