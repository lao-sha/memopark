# 墓地详情页样式对照表

## 一、组件替换对照

### 1. 页面容器

| 原样式 | 新样式类名 |
|--------|-----------|
| `<div style={{ padding: 16, maxWidth: 640, margin: '0 auto' }}>` | `<div className="grave-detail-page">` |

### 2. 顶部导航

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<Card>` | `<div className="grave-detail-header">` |
| `<Button>返回</Button>` | `<button className="grave-detail-back-btn">` |
| `<Typography.Title level={4}>` | `<h1 className="grave-detail-title">` |

### 3. 音乐播放器

| 原样式 | 新样式类名 |
|--------|-----------|
| `<Card style={{ marginBottom: 16 }}>` | `<div className="grave-detail-music">` |

### 4. 标签页容器

| 原样式 | 新样式类名 |
|--------|-----------|
| `<Tabs>` 默认样式 | `<div className="grave-detail-tabs"><Tabs>` |

### 5. 逝者信息卡片

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<Card>` | `<div className="deceased-info-card">` |
| 卡片标题 | `<div className="deceased-info-header">` |
| 卡片内容 | `<div className="deceased-info-body">` |

### 6. 相册网格

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<List grid={{ column: 3 }}>` | `<div className="album-grid">` |
| `<List.Item>` | `<div className="album-item">` |

### 7. 视频列表

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<List.Item>` | `<div className="video-list-item">` |

### 8. 文章列表

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<Card>` | `<div className="article-list-item">` |

### 9. 底部操作按钮

| 原组件 | 新HTML结构 |
|--------|-----------|
| `<Space>` | `<div className="grave-detail-actions">` |
| `<Button type="primary">` | `<button className="grave-action-btn">` |
| `<Button>` | `<button className="grave-action-btn grave-action-btn-secondary">` |

## 二、常用样式类速查

## 五、间距规范

使用8px基准间距系统：

| 间距 | 用途 |
|------|------|
| 8px | 小间距（图标与文字） |
| 12px | 中等间距（卡片内边距） |
| 16px | 大间距（容器边距） |
| 24px | 超大间距（区块间距） |

const AlbumGrid = ({ photos }) => {
