### 第一步：首页展示测试

#### 1.1 访问首页

**操作：**
- 浏览器打开：`http://localhost:3000/#/`
- 或访问：`http://localhost:3000/#/home`

**预期结果：**

- [ ] Hero 区高度适中（480px）

## ✅ 完整流程测试

### 正常创建流程

1. **访问首页**
   - [ ] 首页正确加载
   - [ ] 所有区域正常显示

2. **点击创建**
   - [ ] 点击"创建纪念馆"卡片
   - [ ] 正确跳转到 `#/grave/create`

3. **查看创建页面**
   - [ ] 页面完整加载
   - [ ] 进度条显示步骤 1
   - [ ] 所有提示信息显示
   - [ ] 表单正确显示
   - [ ] 使用说明完整

4. **填写表单**
   - [ ] 输入逝者姓名
   - [ ] （可选）输入园区 ID
   - [ ] 字段验证正常

5. **提交创建**
   - [ ] 点击"创建纪念馆"按钮
   - [ ] 密码弹窗弹出
   - [ ] 弹窗样式正确

6. **输入密码**
   - [ ] 输入正确密码（≥ 8 位）
   - [ ] 点击"确认创建"
   - [ ] 显示加载状态

7. **等待确认**
   - [ ] 显示"正在提交交易..."
   - [ ] 超过 8 秒显示"连接较慢"提示
   - [ ] 进度条更新到步骤 2

8. **创建成功**
   - [ ] 显示成功提示（含交易哈希）
   - [ ] 进度条更新到步骤 3
   - [ ] 2 秒后自动跳转

9. **自动跳转**
   - [ ] URL 变为 `#/grave/my`
   - [ ] 进入"我的纪念馆"页面
   - [ ] 能看到新创建的纪念馆
