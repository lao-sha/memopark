# Stardust 新版首页设计说明

## 概述

新版首页 (`ModernHomePage`) 采用现代化设计理念，参考「云上思念」网站的优秀设计元素，为用户提供更直观、美观、易用的界面体验。

## 设计理念

### 1. 视觉设计
- **Hero 区域**: 顶部横幅展示轮播图或默认欢迎信息，高度 460px（移动端自适应）
- **三色渐变背景**: 融合深金色、墨绿色、朱红色三种主题色的渐变效果
- **卡片式布局**: 统一的圆角（16px）、阴影效果、边框装饰
- **响应式设计**: 适配移动端、平板、桌面端，断点完善优化
- **装饰元素**: 添加顶部装饰条、标题左侧装饰条等细节

### 2. 色彩系统
遵循项目定义的纪念主题配色，参考云上思念的庄重典雅风格：
- 主色：深金色 `#B8860B` - 庄重、永恒
- 辅色：墨绿色 `#2F4F4F` - 生命、希望
- 强调色：朱红色 `#DC143C` - 祭品、献花
- 背景：米白色 `#F5F5DC` - 温暖、怀念
- 文字：深灰 `#708090` - 柔和、易读

### 3. 交互体验
- **悬停动效**: 卡片悬停时上浮（8-10px）、缩放（1.02-1.03）、增强阴影
- **平滑过渡**: 所有交互均使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动函数
- **渐进加载**: 使用错峰渐显动画（fadeIn）增强视觉层次感
- **脉冲动画**: Hero 区按钮添加脉冲效果，吸引用户注意
- **数字滚动**: 统计数据悬停时放大效果

## 技术实现

### 文件结构
```
stardust-dapp/src/features/home/

├── ModernHomePage.css      # 样式文件（优化后）

```

### 路由配置
- 主路由：`#/home` 或 `#/`
- 旧版首页：`#/home-old`

### 响应式断点 - 优化后
- **移动端**：`< 576px`
  - Hero 高度：280px
  - 标题：30px
  - 副标题：18px
  - 统计卡片上移：-20px
  - 单列布局
  
- **平板**：`576px - 768px`
  - Hero 高度：320px
  - 标题：36px
  - 副标题：20px
  - 统计卡片上移：-30px
  - 2 列布局
  
- **桌面**：`> 768px`
  - Hero 高度：460px
  - 标题：54px
  - 副标题：26px
  - 统计卡片上移：-30px
  - 4 列布局

### 动画效果 - 优化后
1. **页面淡入动画**（pageEnter）
   - 整个页面淡入：0.5s
   - 缓动函数：ease-out

2. **卡片渐显动画**（fadeIn）
   - 从下方 30px 淡入 + 缩放（0.95 → 1）
   - 错峰延迟：
     - 统计卡片：0.05s, 0.1s, 0.15s, 0.2s
     - 快捷操作：0.1s, 0.2s, 0.3s, 0.4s
     - 特色功能：0.15s, 0.25s, 0.35s, 0.45s
   - 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)

3. **脉冲动画**（pulse）
   - Hero 区按钮 2s 循环
   - 缩放：1 → 1.05 → 1
   - 阴影扩散效果

4. **悬停动效** - 增强版
   - 统计卡片：上浮 -8px + 缩放 1.02
   - 快捷操作：上浮 -10px + 缩放 1.03 + 图标旋转 -5deg
   - 特色功能：上浮 -8px + 缩放 1.02 + 图标旋转 -8deg + 缩放 1.15
   - 探索发现：上浮 -4px
   - 过渡时间：0.3s - 0.4s
   - 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)

## 使用说明

### 1. 访问首页
- 登录后自动跳转到新版首页
- 或直接访问 `#/home`

### 2. 快捷操作
点击快捷操作卡片可快速跳转到对应功能：
- **创建纪念馆**：`#/grave/create`
- **我要供奉**：`#/offerings/timeline`
- **探索纪念馆**：`#/deceased/list`
- **我的纪念馆**：`#/grave/my`

### 3. 轮播图管理
轮播图数据从链上读取（`memoGrave.carousel()`），需要管理员权限配置。

### 4. 统计数据
统计数据会定期从链上更新，包括：
- 纪念馆总数
- 供奉次数
- 活跃用户数
- 记忆条目数

## 兼容性说明

- 旧版首页仍然可用：`#/home-old`

   - 统计数据建议缓存
   - 长列表需要分页加载

2. **无障碍支持**
   - 所有交互元素支持键盘导航
   - 图片添加 alt 文本
   - 合理的语义化 HTML

3. **移动端体验**
   - 触摸区域足够大（最小 44px）
   - 避免悬停效果在移动端失效
   - 优化移动端加载速度
