# 聊天页面更新 - 添加通讯录入口

## 🎯 更新内容

### 1. 智能群聊页面 (`#/smart-chat`)
**位置**: 右上角
**按钮**: 通讯录图标（ContactsOutlined）
**功能**: 点击跳转到通讯录页面 `#/contacts`

**视觉效果**:
```
┌──────────────────────────────────┐
│ 📱 智能群聊     📇 [返回]       │
│ 端到端加密 • 量子抗性安全         │
└──────────────────────────────────┘
```

### 2. 一对一聊天页面 (`#/chat`)
**位置**: 右上角
**按钮**: 通讯录图标（ContactsOutlined）+ 新建对话图标（PlusOutlined）
**功能**:
- 通讯录按钮：跳转到通讯录页面 `#/contacts`
- 新建对话按钮：创建新会话

**视觉效果**:
```
┌──────────────────────────────────┐
│ 💬 私信         📇 ➕            │
│ 端到端加密聊天                   │
└──────────────────────────────────┘
```

## 📐 设计规范

### 按钮样式
```css
className="p-2 bg-white/20 backdrop-blur-sm hover:bg-white/30 rounded-xl transition-all"
```

**特点**:
- 圆角: `rounded-xl` (12px)
- 背景: 半透明白色 20% 透明度
- 悬停: 30% 透明度
- 毛玻璃效果: `backdrop-blur-sm`
- 过渡动画: `transition-all`

### 图标大小
- 智能群聊: `text-lg` (18px)
- 一对一聊天: `text-xl` (20px)

### 布局结构
```jsx
<div className="flex items-center gap-2">
  <button title="通讯录">
    <ContactsOutlined />
  </button>
  <button title="新建对话">
    <PlusOutlined />
  </button>
</div>
```

## 🔗 页面跳转

### 跳转方式
```javascript
onClick={() => {
  window.location.hash = '#/contacts'
}}
```

### 路由配置
确保 `routes.tsx` 中已配置通讯录路由：
```typescript
{
  match: h => h === '#/contacts',
  component: lazy(() => import('./features/contacts/ContactsPage'))
}
```

## 💡 用户体验

### 交互流程
1. **查看通讯录**: 点击通讯录图标 → 跳转到通讯录管理页面
2. **快速发起聊天**: 在通讯录中选择联系人 → 发起对话
3. **返回聊天**: 使用浏览器后退或导航返回聊天页面
