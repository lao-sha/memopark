## 📝 使用场景

### 场景 1：审批做市商申请

**角色**：Council 成员

1. 访问 `/approval` 页面
2. 查看待审批的申请列表
3. 点击"发起提案"
4. 等待其他成员投票
5. 达到 2/3 阈值后点击"执行"

**完整流程**：

```
申请提交 → 发起提案 → Council 投票 → 执行提案 → 做市商激活
```

---

### 场景 2：创建挂单

**角色**：已批准的做市商

1. 访问 `/listing` 页面
2. 确认账户是做市商
3. 点击"创建挂单"
4. 填写挂单信息：
   - 交易方向：卖出 MEMO
   - 价差：100 bps (1%)
   - 最小数量：100 MEMO
   - 最大数量：1000 MEMO
   - 总库存：5000 MEMO
   - 过期时间：当前区块 + 10000
5. 提交挂单

**注意**：创建挂单时会锁定相应的库存。

## 🐛 故障排除

### 问题：链端连接失败

```bash
# 检查链端是否运行
ps aux | grep stardust-node

# 检查端口
netstat -an | grep 9944

# 重启链端
pkill -f stardust-node
./target/release/stardust-node --dev --rpc-cors all
```

### 问题：钱包连接失败

1. 确认已安装 Polkadot.js Extension
2. 在扩展中允许网站访问
3. 刷新页面重试

### 问题：交易失败

1. 查看浏览器控制台错误信息
2. 检查账户余额是否充足
3. 确认账户权限是否正确
