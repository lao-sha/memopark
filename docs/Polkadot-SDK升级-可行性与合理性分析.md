| 变更类型 | 概率 | 影响范围 | 修复难度 |
|----------|------|----------|----------|
| Breaking Changes | 低 | Trait 签名变更 | 中等 |
| Deprecations | 中 | 警告，仍可用 | 低 |
| New Features | 高 | 可选使用 | 无 |
| Bug Fixes | 高 | 行为修正 | 低 |

#### 常见 API 变更模式

**模式 1: 函数签名变更**


**预估影响**: 5-10 处修改


**修复策略**: 实现新 Trait
