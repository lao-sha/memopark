## 💡 技术亮点汇总

### 联盟计酬系统
- **即时/周结算双模式**: 灵活切换，满足不同场景
- **活跃度管理**: 延长/到期机制，精细化运营
- **15层推荐链**: 深度分销，循环检测防作弊
- **一次绑定**: 推荐人终身有效，关系稳定

### OTC/Bridge交易系统
- **分阶段审核**: 锁定押金→提交资料→治理审核
- **数据脱敏**: 姓名/身份证/生日链上仅存脱敏版
- **OCW自动验证**: 查询TRON链验证转账，无需人工
- **多路分账**: 买家88% + 联盟10% + 平台2%
- **信用保护**: 买家/做市商双向信用评估
- **首购资金池**: 7天冷却期+最小保留余额保护

### 核心业务系统
- **多路分账路由**: 主题/联盟/系统费用智能分配
- **低耦合设计**: GraveInspector Trait解耦墓地/逝者
- **准入策略保护**: 控制安葬权限
- **IPFS自动Pin**: 确保媒体内容持久化
- **会员折扣**: 动态折扣，激励会员消费

### 价格与统计系统
- **滑动窗口聚合**: 最近100万MEMO交易
- **双市场加权**: OTC+Bridge价格加权平均
- **冷启动保护**: 初期使用默认价格，单向锁定退出
- **去重保护**: 基于tx_key防止重复计数
