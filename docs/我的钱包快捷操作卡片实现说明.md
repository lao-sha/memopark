# 我的钱包快捷操作卡片实现说明

## 功能概述

在"我的钱包"页面,将原来菜单列表中的"转账"和"收款"两个菜单项,改为独立的快捷操作卡片区域,并扩展为四个入口:
- **转账**
- **收款**
- **购买MEMO**
- **兑换MEMO**

采用两行两列的网格布局,提供更直观、更便捷的操作入口。

## 实现位置

- **文件**: `/home/xiaodong/文档/memopark/memopark-dapp/src/features/profile/MyWalletPage.tsx`
- **位置**: 头部信息卡片和菜单列表之间

## 布局设计

### 网格布局
使用 CSS Grid 实现两行两列布局:
```css
{
  display: 'grid',
  gridTemplateColumns: '1fr 1fr',  /* 两列等宽 */
  gap: '12px',                      /* 卡片间距 12px */
  padding: '0 16px',                /* 左右内边距 */
}
```

### 卡片排列顺序
```
第一行: [转账]     [收款]
第二行: [购买MEMO] [兑换MEMO]
```

## 交互效果

### 悬停效果
使用 `onMouseEnter` 和 `onMouseLeave` 实现:
```typescript
onMouseEnter={(e) => {
  e.currentTarget.style.transform = 'translateY(-4px)';
  e.currentTarget.style.boxShadow = '0 4px 16px rgba(102, 126, 234, 0.2)';
}}
onMouseLeave={(e) => {
  e.currentTarget.style.transform = 'translateY(0)';
  e.currentTarget.style.boxShadow = '0 2px 8px rgba(0, 0, 0, 0.06)';
}}
```

**效果**:
- 向上移动 4px (`translateY(-4px)`)
- 阴影从浅灰色变为紫色调,范围增大
- 过渡动画: `transition: 'all 0.3s'`

### 点击反馈
- 每个卡片都有明确的点击处理函数
- 点击后执行相应的跳转或弹窗操作
- 提供即时的视觉和功能反馈
